Este projeto é um sistema de biblioteca desenvolvido em Python com o objetivo de colocar em prática os principais conceitos aprendidos na disciplina. Por meio de um menu interativo, o usuário pode cadastrar livros, consultar informações por código ou autor, alterar e remover registros, listar os livros em ordem alfabética e controlar empréstimos e devoluções. Todos os dados são armazenados em listas durante a execução do programa, para simular o gerenciamento de uma pequena biblioteca.
